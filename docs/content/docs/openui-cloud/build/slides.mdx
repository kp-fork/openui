title: Slides
description: "Generate presentations with well-designed templates, manual editing, an interactive viewer, and PowerPoint export."
---

Using OpenUI Cloud, agents can generate presentations from a prompt or conversation with well-designed templates for titles, charts, images, comparisons, and other common slide layouts.

Generated decks open in an interactive viewer where you can review the presentation, edit slides manually, and export the finished deck to PowerPoint. Decks can also be refined through conversation, and every version is stored automatically.
